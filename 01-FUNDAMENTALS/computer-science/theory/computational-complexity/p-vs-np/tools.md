## BEGINNER

### Complexity Visualizer
Author: Various open-source contributors
Year: 2024
Link: https://github.com/topics/complexity-visualizer
Description: Visual tools for understanding algorithm complexity and Big-O notation.
Why included: Makes complexity analysis tangible and interactive.

### Algorithm Analyzer
Author: David Galles
Year: 2024
Link: https://www.cs.usfca.edu/~galles/visualization/Algorithms.html
Description: Interactive visualizations of classic algorithms with complexity analysis.
Why included: Learn by watching algorithms execute with real-time complexity tracking.

## INTERMEDIATE

### SAT Solvers (Educational Versions)
Author: Various researchers
Year: 2024
Link: https://github.com/topics/sat-solver
Description: Implementations of SAT solving algorithms (MiniSat, Glucose, CaDiCaL).
Why included: Study the canonical NP-complete problem through implementation.

### NP-Completeness Proof Generator
Author: Various
Year: 2023
Link: https://github.com/search?q=np-completeness+proof&type=repositories
Description: Tools for automatically generating reduction-based NP-completeness proofs.
Why included: Automate the mechanical parts of proving NP-completeness.

## ADVANCED

### Complexity Prover (Automated Complexity Analysis)
Author: INRIA, various universities
Year: 2023-2024
Link: https://github.com/topics/complexity-analysis+proof
Description: Tools for automated program complexity analysis and certification.
Why included: Formally verify complexity bounds using proof assistants.

### Constraint Programming Solver
Author: Google OR-Tools and others
Year: 2024
Link: https://developers.google.com/optimization
Description: Industrial-strength solvers for constraint satisfaction and optimization problems.
Why included: See how real-world NP-hard problems are tackled pragmatically.

Author: IBM, Gurobi Optimization
Year: 2000s-present
Link: https://www.ibm.com/products/ilog-cplex-optimization-studio
Description: Industry-standard optimization solvers with academic licenses.
Why included: Gold standard for practical NP-hard problem solving.

### The Algorithm Design Manual (Online Companion)
Author: Steven Skiena
Year: 2024
Link: https://www3.cs.stonybrook.edu/~skiena/algorist/
