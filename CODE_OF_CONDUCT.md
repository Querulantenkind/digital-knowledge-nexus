# Code of Conduct

## Our Commitment

Digital Knowledge Nexus is a technical knowledge repository. We are committed to maintaining a professional, inclusive, and harassment-free environment for all contributors regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

---

## Expected Behavior

All contributors are expected to:

**Be Professional**
- Focus on technical merit of contributions
- Provide constructive feedback
- Accept criticism gracefully
- Stay on topic in discussions

**Be Respectful**
- Disagree without being disagreeable
- Assume good faith in others
- Respect different viewpoints and experiences
- Use welcoming and inclusive language

**Be Collaborative**
- Help newcomers understand the structure
- Explain decisions clearly
- Work towards consensus on disputed issues
- Share knowledge freely

**Be Accurate**
- Verify information before contributing
- Admit mistakes and correct them
- Cite sources properly
- Do not plagiarize

---

## Unacceptable Behavior

The following behaviors are unacceptable:

**Harassment**
- Offensive comments related to personal characteristics
- Deliberate intimidation, stalking, or following
- Unwelcome sexual attention or advances
- Publishing others' private information without permission

**Disruption**
- Trolling or deliberately inflammatory comments
- Sustained disruption of discussions
- Spam or promotional content
- Deliberate misinformation

**Abuse**
- Personal attacks or insults
- Threats of violence
- Encouraging self-harm
- Doxxing or privacy violations

**Academic Dishonesty**
- Plagiarism
- Misattribution
- Intentional misinformation
- Citation manipulation

---

## Scope

This Code of Conduct applies to:
- All repository spaces (issues, pull requests, discussions)
- Official communication channels
- Events or meetings related to this project
- Public spaces when representing the project

---

## Enforcement

### Reporting

If you experience or witness unacceptable behavior:

1. **For immediate threats:** Contact local authorities
2. **For Code of Conduct violations:** 
   - Report via GitHub's reporting tools
   - Email repository maintainers with details
   - Include links, screenshots, or evidence

All reports will be reviewed and investigated confidentially.

### Consequences

Depending on severity, violations may result in:

**First Offense (Minor):**
- Private warning
- Required apology
- Temporary comment restriction

**Second Offense (Minor) or First Offense (Moderate):**
- Temporary ban (7-30 days)
- Public warning
- Contribution restrictions

**Severe or Repeated Violations:**
- Permanent ban from the repository
- Report to GitHub
- Report to relevant authorities (if illegal activity)

### Appeals

If you believe enforcement action was unjust:
1. Email maintainers within 7 days
2. Provide context and evidence
3. Await review (14 days)
4. Accept final decision

## Technical Disputes

This Code of Conduct covers behavior, not technical disagreements.

**For technical disputes:**
- Follow GOVERNANCE.md processes
- Focus on facts and evidence
- Accept maintainer decisions
- Do not make disputes personal

**Disagreements about:**
- Category placement → Refer to TAXONOMY.md
- Quality standards → Refer to QUALITY_CRITERIA.md
- Formatting → Refer to STYLE_GUIDE.md

## Maintainer Responsibilities

Maintainers will:
- Model expected behavior
- Enforce this Code fairly and consistently
- Recuse themselves from conflicts of interest
- Protect reporter confidentiality
- Document enforcement actions

Maintainers who violate this Code face stricter consequences including removal from maintainer role.

---

## Questions

Unclear about whether behavior is acceptable? Ask yourself:
- Would I say this in a professional workplace?
- Does this contribute to technical discussion?
- Would I want this said to/about me?
- Is this relevant to the repository's purpose?

When in doubt, err on the side of professionalism.
