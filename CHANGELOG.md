## Format

### Change Types
- **Added**: New features, categories, or documentation
- **Changed**: Changes to existing functionality or structure
- **Deprecated**: Features or categories marked for removal
- **Removed**: Deleted features or categories
- **Fixed**: Bug fixes or corrections
- **Security**: Security-related changes

### What Goes in This Changelog
**DO include:**
- New domains or major categories
- Structural reorganizations
- Policy changes
- Documentation updates (major)
- Format file changes
- Template updates
- Security fixes

**DON'T include:**
- Individual resource additions (tracked in Git)
- Typo fixes
- Broken link fixes
- Minor documentation tweaks
- Routine maintenance

## Version Numbering

Following Semantic Versioning (SemVer):

**Major version (X.0.0)**: Breaking changes
- Structural reorganization
- Category renames or moves
- Format file changes
- Breaking policy changes

**Minor version (0.X.0)**: New features
- New major categories
- New documentation
- New format types (unlikely)
- Policy additions

**Patch version (0.0.X)**: Fixes
- Documentation corrections
- Template fixes
- Minor policy clarifications
