## BEGINNER

### Memory Visualization Tool (Villanelle)
Creator: David Benson
Year: 2023
Platform: Web-based
Link: https://www.villanelleapp.com/
Type: Interactive visualizer
Prerequisites: Understanding of C/basic programming
Description: Step-through visualizer showing memory layout, pointer operations, and allocation in C programs.
Why included: Hands-on exploration of memory concepts; makes abstract memory operations concrete.

### C Memory Explorer (freeCodeCamp)
Creator: freeCodeCamp
Year: 2024
Platform: Web browser
Link: https://www.freecodecamp.org/
Type: Interactive tutorial with embedded exercises
Prerequisites: Basic C knowledge
Description: Guided exercises exploring pointers, arrays, memory allocation with immediate feedback.
Why included: Structured progression from basics to practical memory management.

## INTERMEDIATE

### Memory Profiling Playground (Valgrind Tutorial)
Creator: Julian Seward & team
Year: Ongoing
Platform: Linux/Unix terminals
Link: https://www.valgrind.org/
Type: Interactive tool with exercises
Prerequisites: C/C++ programming, Linux experience
Description: Hands-on exercises using Valgrind to detect memory leaks, improper allocations, and undefined behavior.
Why included: Learning to use industry-standard debugging tools; practical skills for production development.

### Rust Playground (Official)
Creator: Rust Foundation
Year: 2023
Platform: Web-based & offline
Link: https://play.rust-lang.org/
Type: Interactive code editor with visualization
Prerequisites: Basic programming knowledge
Description: Experiment with Rust ownership and borrowing concepts in browser with immediate compilation feedback.
Why included: Explore memory safety paradigm without installation; excellent for understanding Rust's approach.
