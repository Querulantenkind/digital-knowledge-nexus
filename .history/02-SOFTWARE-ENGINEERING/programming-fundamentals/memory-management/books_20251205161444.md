## BEGINNER

### The C Programming Language
Author: Brian W. Kernighan, Dennis M. Ritchie
Year: 1988 (2nd Edition)
Publisher: Prentice Hall
ISBN: 978-0131103627
Link: https://www.pearson.com/en-us/subject-catalog/p/c-programming-language-the-2nd-edition/P200000003487/9780131103627
Description: Classic introduction to C with fundamental coverage of pointers, arrays, and manual memory management.
Why included: Canonical text for learning memory fundamentals; authors are creators of C.

### Effective C: An Introduction to Professional C Programming
Author: Robert C. Seacord
Year: 2020
Publisher: No Starch Press
ISBN: 978-1718501041
Link: https://nostarch.com/effective-c
Description: Modern guide to C best practices with emphasis on memory safety, allocation patterns, and common pitfalls.
Why included: Contemporary approach to C memory management with practical security and reliability considerations.

---

## INTERMEDIATE

### Computer Systems: A Programmer's Perspective
Author: Randal E. Bryant, David R. O'Hallaron
Year: 2015 (3rd Edition)
Publisher: Pearson
ISBN: 978-0134092669
Link: https://csapp.cs.cmu.edu/
Description: Comprehensive treatment of memory hierarchy, virtual memory, dynamic memory allocation, and garbage collection from system perspective.
Why included: Essential systems perspective; connects memory management to hardware, OS, and application layers.

### Programming in Rust (with Memory Management Focus)
Author: Steve Klabnik, Carol Nichols
Year: 2023
Publisher: No Starch Press
ISBN: 978-1492052593
Link: https://doc.rust-lang.org/book/
Description: Official Rust book with deep focus on ownership, borrowing, and memory safety without garbage collection.
Why included: Modern approach to memory safety; demonstrates how language design enforces memory correctness.

### Advanced Memory Management
Author: Michael Spertus
Year: 2024
Publisher: Manning Publications
Link: https://www.manning.com/books/advanced-memory-management
Description: Comprehensive treatment of modern memory management techniques including smart pointers, RAII, and memory pools.
Why included: Contemporary best practices across multiple languages and paradigms.

---

## ADVANCED

### Garbage Collection: Algorithms for Automatic Dynamic Memory Management
Author: Richard Jones, Antony Hosking, J. Eliot B. Moss
Year: 2011
Publisher: CRC Press
ISBN: 978-1420082791
Link: https://www.gchandbook.org/
Description: Definitive reference on garbage collection algorithms including mark-sweep, copying collection, generational collection, and concurrent algorithms.
Why included: Comprehensive technical reference; essential for understanding modern language runtimes.

### The Garbage Collection Handbook: The Art of Automatic Memory Management
Author: Richard Jones, Antony Hosking, J. Eliot B. Moss
Year: 2012
Publisher: CRC Press
ISBN: 978-1420082791
Description: Practical guide to implementing garbage collectors with analysis of trade-offs and performance characteristics.
Why included: Technical depth combined with practical implementation guidance.

---

## CLASSIC

### A List-Processing Language for Symbolic Computation
Author: John McCarthy
Year: 1960
Venue: ACM Communications
Description: Original LISP paper introducing automatic memory management and garbage collection concepts.
Why included: Historical foundation; introduced GC to computer science.

### Automatic Memory Management in the GNU Emacs Lisp Interpreter
Author: Richard M. Stallman
Year: 1989
Description: Early practical implementation of memory management in a Lisp environment.
Why included: Historical influence on modern garbage collection practice.
