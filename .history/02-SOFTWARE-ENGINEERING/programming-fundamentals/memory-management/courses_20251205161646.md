## BEGINNER

### Introduction to Memory Management (freeCodeCamp)
Instructor: Various instructors
Year: 2024
Platform: freeCodeCamp (YouTube)
Link: https://www.freecodecamp.org/
Duration: ~2-4 hours
Prerequisites: Basic C or C++ knowledge
Description: Visual introduction to memory management concepts including stack, heap, pointers, and allocation with practical demonstrations.
Why included: Accessible, free, visual explanations suitable for beginners entering systems programming.

### Systems Programming & Operating Systems (University at Buffalo)
Instructor: Ethan L. Miller, Steven Swanson, et al.
Year: 2023
Platform: Coursera
Link: https://www.coursera.org/learn/system-design
Duration: 5-6 weeks, ~5-7 hours/week
Prerequisites: C/C++ programming experience
Description: Introduction to systems programming with focus on memory management, address spaces, and process memory layout.
Why included: University-quality introduction connecting memory management to OS fundamentals.

## INTERMEDIATE

### Computer Systems: A Programmer's Perspective (CMU 15-213)
Instructor: Randal E. Bryant, David R. O'Hallaron
Year: Ongoing
Platform: CMU (lecture videos available)
Link: https://csapp.cs.cmu.edu/
Duration: 15 weeks, ~10-12 hours/week
Prerequisites: C programming, basic systems knowledge
Description: Graduate-level course covering memory hierarchy, caching, dynamic memory allocation, virtual memory, and linking.
Why included: Industry-standard systems course; provides comprehensive understanding of how memory works from hardware to application level.

### Advanced C++: Memory Management & Modern Practices (Udemy/Pluralsight)
Instructor: Scott Meyers, Herb Sutter
Year: 2023
Platform: Pluralsight/Various
Link: https://www.pluralsight.com/courses/cplusplus-memory-management
Duration: Self-paced, ~8-12 hours
Prerequisites: Intermediate C++ knowledge
Description: Deep dive into C++ memory management including smart pointers, RAII, move semantics, and resource management patterns.
Why included: Modern C++ perspective on safe and efficient memory management.

## ADVANCED

### Programming Language Implementation (UC Berkeley CS164)
Instructor: Various
Year: Ongoing
Platform: UC Berkeley OpenCourseWare
Link: https://inst.eecs.berkeley.edu/~cs164/sp20/
Duration: 14 weeks
Prerequisites: Strong programming background, compilers knowledge
Description: Covers memory management implementation in language runtimes, including garbage collection algorithms and memory layout.
Why included: Language implementation perspective on memory management design decisions.

### Garbage Collection Algorithms (MIT Advanced Topics)
Instructor: Martin C. Rinard, others
Year: 2020
Platform: MIT OpenCourseWare
Link: https://ocw.mit.edu/
Duration: Self-paced
Prerequisites: Graduate-level systems knowledge
Description: Advanced course on garbage collection algorithm design, performance analysis, and concurrent/parallel collection.
Why included: Research-level treatment of GC theory and practice.

## RECENT (2023+)

### Rust Memory Management (Official Rust Course)
Instructor: The Rust Project
Year: 2024
Platform: Official Rust Documentation
Link: https://doc.rust-lang.org/book/ch04-00-understanding-ownership.html
Duration: Self-paced, ~2-3 hours
Prerequisites: Basic programming knowledge
Description: Modern approach to memory safety through ownership and borrowing, avoiding manual management and garbage collection.
Why included: Exemplar of memory safety without GC; influential design that other languages are adopting.
