## BEGINNER

### Introduction to Algorithms (CLRS) - Chapter 10 (Elementary Data Structures)
Author: Thomas H. Cormen, Charles E. Leiserson, Ronald L. Rivest, Clifford Stein
Year: 2009 (3rd Edition)
Publisher: MIT Press
ISBN: 978-0262033848
Link: https://mitpress.mit.edu/9780262033848/introduction-to-algorithms/
Description: Clear introduction to stacks and queues including operations, implementation with arrays/linked lists, and applications.
Why included: Foundational reference; simple yet rigorous treatment of fundamental data structures.

### Data Structures and Algorithms in Java
Author: Michael T. Goodrich, Roberto Tamassia, Michael H. Goldwasser
Year: 2014 (6th Edition)
Publisher: Wiley
ISBN: 978-1118771334
Link: https://www.wiley.com/
Description: Practical treatment of stacks and queues with complete Java implementations and visual representations.
Why included: Accessible introduction with working code and diagrams.

## INTERMEDIATE

### Algorithms in C (Chapters on Linked Lists and Basic Structures)
Author: Robert Sedgewick
Year: 1998 (3rd Edition)
Publisher: Addison-Wesley
ISBN: 978-0201314653
Link: https://algs4.cs.princeton.edu/
Description: Elegant implementations of stacks and queues with empirical performance analysis.
Why included: Sedgewick's pedagogical approach; connects theory to working implementations.

## ADVANCED

### Advanced Stack/Queue Applications and Analysis
Author: Various (research papers)
Year: 2010s-2020s
Description: Research on concurrent stacks/queues, lock-free implementations, and specialized variants.
Why included: Production-oriented perspective on advanced data structure design.

## CLASSIC

### Fundamental Theory of Data Structures
Author: Alfred V. Aho, John E. Hopcroft, Jeffrey D. Ullman
Year: 1974
Publisher: Addison-Wesley
ISBN: 978-0201000627
Description: Foundational textbook establishing data structure theory and stack/queue fundamentals.
Why included: Historical importance; seminal work in data structures theory.
