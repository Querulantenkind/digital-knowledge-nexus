## BEGINNER

### Introduction to Data Structures (CS50)
Instructor: David J. Malan (Harvard)
Year: Ongoing
Platform: edX / Harvard
Link: https://cs50.harvard.edu/
Duration: Week 5-6, ~5-6 hours
Prerequisites: None
Description: Clear visual introduction to stacks and queues with applications in computer science.
Why included: Excellent pedagogical approach; makes abstract structures concrete with demonstrations.

### Algorithms, Part I (Coursera)
Instructor: Robert Sedgewick, Kevin Wayne
Year: Ongoing
Platform: Coursera
Link: https://www.coursera.org/learn/algorithms-part1
Duration: Early weeks, ~5-6 hours
Prerequisites: Basic programming
Description: Fundamental treatment of stacks and queues with implementations and applications.
Why included: World-class teaching; clear explanations with working code examples.

## RECENT (2023+)

### Lock-Free Data Structures and Concurrent Programming
Instructor: Research faculty
Year: 2023-2024
Platform: Conference workshops / Research seminars
Description: Modern approaches to concurrent stacks and queues for parallel systems.
Why included: Contemporary systems programming challenges; shows evolution of data structure design.
