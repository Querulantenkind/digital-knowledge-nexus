## BEGINNER

### Algorithms, Part I (Coursera)
Instructor: Robert Sedgewick, Kevin Wayne (Princeton)
Year: Ongoing
Platform: Coursera
Link: https://www.coursera.org/learn/algorithms-part1
Duration: Week on hash tables ~6 hours
Prerequisites: Basic programming
Description: Introduction to hash tables including open and closed hashing with visual algorithm demonstrations.
Why included: Excellent teaching quality; visualizations clarify collision handling strategies.

### Data Structures Fundamentals (freeCodeCamp)
Instructor: Various
Year: 2023
Platform: freeCodeCamp / YouTube
Link: https://www.freecodecamp.org/
Duration: 2-3 hours (hash tables section)
Prerequisites: Basic programming
Description: Accessible introduction to hash tables with practical implementation examples.
Why included: Free, clear explanations suitable for beginners.
