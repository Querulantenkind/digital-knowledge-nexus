### Binary Search Tree Visualizer
Creator: Algorithm educators
Year: 2015-Present
License: Educational use
Platform: Web-based

Language: JavaScript
Description: Interactive visualization of binary search tree operations.

Creator: Algorithm educators
Year: 2010-Present
License: Open source (varies)
Platform: Python
Link: https://github.com/topics/bst
Language: Python
Description: Simple to medium complexity BST implementations for learning.
