## BEGINNER

### VisuAlgo Sorting Visualizer
Creator: Steven Halim (National University of Singapore)
Year: 2011-Present
License: Creative Commons
Platform: Web-based (cross-platform)
Link: https://visualgo.net/en/sorting
Language: JavaScript/HTML5
Description: Interactive visualization tool for sorting algorithms with step-through execution and custom input support.
Why included: Educational standard; makes algorithm behavior immediately understandable for beginners.

### Python sorted() / sort() Built-ins
Creator: Python Software Foundation
Year: 1991-Present
License: Python License (permissive open source)
Platform: Cross-platform (Python interpreter)
Link: https://docs.python.org/3/
Language: Python
Description: Practical sorting tools built into Python standard library with Timsort implementation.
Why included: Immediate practical use; shows how theoretical algorithms work in production languages.
