## BEGINNER

### Sorting Algorithm Visualizer (VisuAlgo)
Creator: Steven Halim (National University of Singapore)
Platform: Web-based
Link: https://visualgo.net/en/sorting
Type: Interactive visualizer
Prerequisites: None
Description: Step-by-step visualization of sorting algorithms (bubble, selection, insertion, merge, quick, heap sort) with custom input.
Why included: Outstanding visual learning tool; shows algorithm behavior in real-time with adjustable speed and custom data.

### Sorting Algorithm Animations (YouTube + Interactive)
Creator: Various educators
Platform: Web + Video
Link: https://www.youtube.com/results?search_query=sorting+algorithm+animation
Type: Video visualization with interactive elements
Prerequisites: None
Description: Animated demonstrations of sorting algorithms with detailed step-by-step execution.
Why included: Multiple visualization styles help visual and kinesthetic learners understand algorithm flow.
