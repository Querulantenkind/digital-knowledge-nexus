## BEGINNER

### Fibonacci DP Visualizer (Web-based)
Creator: Algorithm educators
Platform: Web browser
Link: https://www.cs.usfca.edu/~galles/visualization/DynamicProgramming.html
Type: Interactive step-through visualizer
Prerequisites: Basic programming knowledge
Description: Visual step-by-step execution of Fibonacci calculation using memoization vs naive recursion.
Why included: Perfect starting point; shows dramatic performance difference between DP and naive approaches.

### LeetCode Dynamic Programming Starter (Interactive Coding)
Creator: LeetCode
Platform: Web-based IDE with problems
Link: https://leetcode.com/explore/featured/card/dynamic-programming/
Type: Interactive coding problems with solutions
Prerequisites: Programming language knowledge
Description: Curated progression of DP problems from easy to medium with discussion and solutions.
Why included: Scaffolded learning; starts with basic patterns and progresses logically.

## INTERMEDIATE

### Edit Distance Visualizer (Interactive)
Creator: Algorithm visualization community
Platform: Web-based tool
Link: https://www.cs.usfca.edu/~galles/visualization/DynamicProgramming.html
Type: Interactive visualization with state table
Prerequisites: Understanding of DP basics
Description: Visual DP table construction for edit distance with step-through execution.
Why included: Shows how DP table builds iteratively; clarifies bottom-up DP approach.

### Longest Common Subsequence (Interactive Workshop)
Creator: Programming educators
Platform: Interactive notebook / web tool
Link: https://interactive-dp.readthedocs.io/
Type: Interactive coding environment
Prerequisites: DP fundamentals
Description: Guided implementation and visualization of LCS algorithm with multiple approaches.
Why included: Hands-on experience comparing top-down vs bottom-up approaches.
