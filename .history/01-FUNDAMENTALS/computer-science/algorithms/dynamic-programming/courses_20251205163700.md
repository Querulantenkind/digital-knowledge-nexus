## BEGINNER

### MIT 6.006 - Introduction to Algorithms (Lecture Sections on DP)
Instructor: Erik Demaine and colleagues
Year: Ongoing (recorded 2005+)
Platform: MIT OpenCourseWare / YouTube
Link: https://ocw.mit.edu/courses/6-006-introduction-to-algorithms-spring-2020/
Description: Core undergraduate algorithm course with clear DP lectures (LCS, matrix chain, knapsack).
Why included: Rigorous introduction with problem sets and lecture notes.

### Coursera - Algorithms, Part I & II (Princeton)
Instructor: Robert Sedgewick and Kevin Wayne
Year: 2011-Present
Platform: Coursera / Princeton
Link: https://www.coursera.org/specializations/algorithms
Description: Practical algorithm implementations; DP appears through classic problem examples.
Why included: Hands-on coding emphasis; ideal for learners who prefer implementation practice.

## INTERMEDIATE

### Stanford Algorithms Specialization (Tim Roughgarden)
Instructor: Tim Roughgarden
Year: 2012-Present
Platform: Coursera / Stanford Online
Link: https://www.coursera.org/specializations/algorithms
Description: Deeper algorithmic techniques including DP optimizations and divide-and-conquer contrasts.
Why included: Bridges theory and practice; good for algorithm design skills.

### UC San Diego - Algorithmic Toolbox
Instructor: Multiple
Year: 2016-Present
Platform: Coursera
Link: https://www.coursera.org/learn/algorithmic-toolbox
Description: Focused problem-solving course with many DP problems and practice exercises.
Why included: Problem-centric; strong for preparing coding interviews.

## ADVANCED

### MIT 6.854 - Advanced Algorithms
Instructor: Erik Demaine
Year: 2014-Present
Platform: MIT OpenCourseWare
Link: https://ocw.mit.edu/courses/6-854-advanced-algorithms-fall-2008/
Description: Graduate-level treatment covering advanced DP topics, approximation, and optimization techniques.
Why included: Research-level depth; useful for specialization and thesis work.

### Parallel and Distributed Algorithms (University Courses)
Instructor: Various
Year: 2015+
Platform: University offerings / MOOCs
Link: https://www.coursera.org/
Description: Explores DP in parallel settings and large-scale optimization.
Why included: Applies DP concepts to modern parallel architectures.

## RECENT (2020+)

### Reinforcement Learning & Dynamic Programming (Deep RL courses)
Instructor: David Silver (DeepMind), others
Year: 2016-Present
Platform: YouTube / University pages
Link: https://www.deepmind.com/learning-resources
Description: Courses connecting DP concepts to reinforcement learning (value iteration, policy iteration).
