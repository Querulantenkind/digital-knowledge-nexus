## BEGINNER

### The String-to-String Correction Problem
Author: Robert A. Wagner and Michael J. Fischer
Year: 1974
Description: Seminal paper introducing the edit distance (Levenshtein distance) metric and dynamic programming solution.
Why included: Foundational work; clear presentation of the problem and classical algorithm.

### Optimal String Alignment Distance
Author: Various algorithm researchers
Description: Papers on variants of edit distance including restricted edit operations and adjacency constraints.
Why included: Understanding different distance metrics and their properties.

## INTERMEDIATE

### Approximate String Matching and Regular Expressions
Author: Aho & Corasick, and others
Year: 1975-1990s
Description: Research on string matching with mismatches and edit distance applications.
Why included: Extending edit distance to pattern matching and fuzzy searching.

### Longest Common Subsequence and Sequence Alignment
Author: Various bioinformatics and computer science researchers
Description: Papers on dynamic programming for sequence comparison and alignment.
Why included: Related problem with deep connections to edit distance; bioinformatics applications.

## ADVANCED

### Fast Approximate String Matching
Author: Myers, Manber, and others
Year: 1990s-2000s
Description: Research on efficient algorithms for approximate string matching using edit distance.
Why included: Optimizations for large-scale string comparison; practical performance techniques.

### Parallel and External Memory Algorithms for Edit Distance
Author: Various systems researchers
Description: Research on scaling edit distance computation to massive datasets.
Why included: Modern applications in bioinformatics, plagiarism detection, and big data.
