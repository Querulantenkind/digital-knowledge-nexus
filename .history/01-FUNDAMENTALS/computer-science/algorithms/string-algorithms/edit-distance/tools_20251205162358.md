## BEGINNER

### Python Difflib Module
Creator: Python Software Foundation
Year: 1991-Present
License: Python License
Platform: Python (cross-platform)
Link: https://docs.python.org/3/library/difflib.html
Language: Python
Description: Built-in Python library for sequence comparison and similarity.
Why included: Practical implementation; immediate use in Python projects.

### LCS (Longest Common Subsequence) Visualizer
Creator: Algorithm educators
Year: 2020s
License: Educational use
Platform: Web-based
Link: https://www.cs.usfca.edu/~galles/visualization/Algorithms.html
Language: JavaScript
Description: Interactive visualization of LCS and edit distance algorithms.
Why included: Visual learning tool; makes dynamic programming approach clear.

## INTERMEDIATE

### rapidfuzz (Fast String Matching)
Creator: Max Bachmann
Year: 2018-Present
License: MIT License (open source)
Platform: Python with C++ backend
Link: https://rapidfuzz.github.io/
Language: Python / C++
Description: High-performance string matching and fuzzy comparison library.
Why included: Production-quality tool; orders of magnitude faster than pure Python.

### Levenshtein Distance Libraries
Creator: Multiple communities
Year: 2000s-Present
License: Various (mostly permissive open source)
Platform: Multiple languages (C, Python, Java, etc.)
Link: https://github.com/topics/levenshtein-distance
Language: Multiple
Description: Optimized implementations of edit distance calculations.
Why included: Practical tool suite; widely used in spell-checking and fuzzy matching.
