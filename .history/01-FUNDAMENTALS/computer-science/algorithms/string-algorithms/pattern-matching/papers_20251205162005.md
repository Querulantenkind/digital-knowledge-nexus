## BEGINNER

### Fast Pattern Matching in Strings
Author: Donald E. Knuth, James H. Morris, Vaughan R. Pratt
Year: 1977
Venue: SIAM Journal on Computing
DOI: 10.1137/0206024
Link: https://doi.org/10.1137/0206024
Description: Original KMP algorithm paper introducing linear-time string matching.
Why included: Seminal work; elegant algorithmic innovation with profound impact.

### An Efficient Algorithm for the Approximate Matching of Strings
Author: Ricardo A. Baeza-Yates
Year: 1989
Venue: Information Processing Letters
DOI: 10.1016/0020-0190(89)90010-3
Link: https://doi.org/10.1016/0020-0190(89)90010-3
Description: Efficient approximate pattern matching algorithms.
Why included: Practical extension of exact matching to real-world applications.

## INTERMEDIATE

### A Fast String Searching Algorithm
Author: Boyer, Robert S., Moore, J. Strother
Year: 1977
Venue: Communications of the ACM
DOI: 10.1145/359842.359859
Link: https://dl.acm.org/doi/10.1145/359842.359859
Description: Boyer-Moore algorithm combining right-to-left scanning and preprocessing.
Why included: Practical algorithm widely used in text editors; excellent algorithmic technique.
