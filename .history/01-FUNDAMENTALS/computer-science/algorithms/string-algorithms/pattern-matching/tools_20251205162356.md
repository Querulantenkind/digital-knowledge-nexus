Creator: Python Software Foundation
Year: 1991-Present
License: Python License
Platform: Python (cross-platform)
Link: https://docs.python.org/3/library/re.html
Language: Python
Description: Standard regex library in Python with extensive pattern matching capabilities.
Why included: Industry-standard tool; powers most practical pattern matching.

Year: 1973-Present
License: GPL (open source)

Link: https://www.gnu.org/software/grep/

Year: 2001-Present
License: BSD License (open source)
Platform: Cross-platform C library

Language: C

Year: 1997-Present
License: BSD License (open source)
Platform: Cross-platform C library
