## BEGINNER

### Algorithms, Part II (Coursera)
Instructor: Robert Sedgewick, Kevin Wayne (Princeton)
Year: Ongoing
Platform: Coursera
Link: https://www.coursera.org/learn/algorithms-part2
Duration: Week 3, ~6 hours
Prerequisites: Algorithms Part I
Description: Introduction to string processing including pattern matching with live demonstrations.
Why included: Excellent pedagogical approach; visual demonstrations of KMP and Boyer-Moore.

### String Matching Algorithms (YouTube Educational)
Instructor: Various educators
Year: 2023
Platform: YouTube
Link: https://www.youtube.com/results?search_query=string+matching+algorithms
Duration: 4-6 hours
Prerequisites: Basic programming
Description: Collection of tutorials explaining pattern matching concepts visually.
Why included: Free, accessible introduction from multiple perspectives.

## INTERMEDIATE

### Advanced Algorithms (Stanford)
Instructor: Tim Roughgarden
Year: Ongoing
Platform: Coursera
Link: https://www.coursera.org/specializations/algorithms
Duration: Week on string algorithms, ~6 hours
Prerequisites: Algorithms fundamentals
Description: Analysis of string matching algorithms with complexity theory and practical considerations.
Why included: Rigorous mathematical treatment from top computer scientist.

## ADVANCED

### String Processing and Genomic Algorithms (MIT)
Instructor: Bonnie Berger, others
Year: Ongoing
Platform: MIT OpenCourseWare
Link: https://ocw.mit.edu/
Duration: Full semester graduate course
Prerequisites: Algorithms, discrete math
Description: Advanced string algorithms with applications to computational biology.
Why included: Research-level treatment; real applications in genomics.

### Pattern Matching in Big Data (Research Seminar)
Instructor: Data systems researchers
Year: 2023-2024
Platform: Conference workshops
