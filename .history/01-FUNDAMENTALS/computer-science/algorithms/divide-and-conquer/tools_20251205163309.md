## BEGINNER

### Python sorted() and list.sort()
Creator: Python Software Foundation
Year: 1991-Present
License: Python License
Platform: Python (cross-platform)
Link: https://docs.python.org/3/howto/sorting.html
Language: Python
Description: Built-in sorting using Timsort (hybrid merge-insertion sort).
Why included: Production-quality divide-and-conquer; most widely used.

### Sorting Visualizer Tools
Creator: Algorithm educators
Year: 2015-Present
License: Open source (various)
Platform: Web-based
Link: https://www.sortvisualizer.com/
Language: JavaScript
Description: Interactive tools for visualizing merge sort, quicksort execution.
Why included: Educational; makes algorithm behavior visible.

## INTERMEDIATE

### C++ std::sort and std::stable_sort
Creator: C++ Standards Committee
Year: 1998-Present
License: Open source (compiler-specific)
Platform: C++ (cross-platform)
Link: https://en.cppreference.com/w/cpp/algorithm/sort
Language: C++
Description: Introsort (hybrid quicksort-heapsort-insertion) implementation.
Why included: Industry standard; excellent performance characteristics.

### NumPy/SciPy FFT Implementation
Creator: NumPy/SciPy community
Year: 2006-Present
License: BSD License
Platform: Python
Link: https://numpy.org/doc/stable/reference/routines.fft.html
Language: Python/C
Description: Fast Fourier Transform using divide-and-conquer (Cooley-Tukey).
Why included: Production FFT; widely used in scientific computing.

## ADVANCED

### FFTW (Fastest Fourier Transform in the West)
Creator: Matteo Frigo, Steven G. Johnson (MIT)
Year: 1997-Present
License: GPL (with commercial licenses)
Platform: C library (cross-platform)
Link: http://www.fftw.org/
Language: C
Description: Highly optimized FFT library using divide-and-conquer with runtime optimization.
Why included: State-of-the-art performance; benchmark standard.

### Intel TBB (Threading Building Blocks)
Creator: Intel
Year: 2006-Present
License: Apache 2.0
Platform: C++ (cross-platform)
Link: https://www.intel.com/content/www/us/en/developer/tools/oneapi/onetbb.html
Language: C++
Description: Parallel sorting and divide-and-conquer patterns for multi-core systems.
Why included: Production parallelism; efficient work stealing.
