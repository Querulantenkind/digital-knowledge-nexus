## BEGINNER

### VisuAlgo Shortest Path Visualizer
Creator: Steven Halim (National University of Singapore)
Year: 2011-Present
License: Creative Commons
Platform: Web-based (cross-platform)
Link: https://visualgo.net/en/sssp
Language: JavaScript/HTML5
Description: Interactive visualization of Dijkstra, Bellman-Ford algorithms with step-through execution.
Why included: Makes shortest path algorithm behavior visible; essential for understanding.

### NetworkX (Python Graph Library)
Creator: Python networkx contributors
Year: 2004-Present
License: BSD License (open source)
Platform: Python ecosystem (cross-platform)
Link: https://networkx.org/
Language: Python
Description: Production-quality graph library with comprehensive shortest path algorithms.
Why included: Industry-standard tool; works with real graph data.
