## BEGINNER

### A Note on Two Problems in Connexion with Graphs
Author: Edsger W. Dijkstra
Year: 1959
Venue: Numerische Mathematik
DOI: 10.1007/BF01386390
Link: https://doi.org/10.1007/BF01386390
Description: Original paper introducing Dijkstra's algorithm for single-source shortest paths, remarkably clear for a foundational work.
Why included: Seminal algorithm; elegant presentation showing power of greedy approach.

### Shortest Paths in Graphs with Unit Weights
Author: Donald B. Johnson
Year: 1977
Venue: Journal of the ACM
DOI: 10.1145/321992.321993
Link: https://dl.acm.org/doi/10.1145/321992.321993
Description: Analysis of shortest path algorithms in special cases with unit edge weights.
Why included: Important special case; shows how problem structure enables efficient solutions.

## INTERMEDIATE

### An Algorithm for the All-Pairs Shortest-Path Problem
Author: Robert W. Floyd
Year: 1962
Venue: IEEE Transactions on Electronic Computers
Description: Floyd-Warshall algorithm for all-pairs shortest paths, simple yet powerful dynamic programming approach.
Why included: Elegant algorithm design; fundamental contribution to graph algorithms.

### Practical Shortest Path Algorithms
Author: Andrew V. Goldberg, Chris Harrelson
Year: 2005
Venue: SODA (Symposium on Discrete Algorithms)
DOI: 10.1145/1070432.1070455
Link: https://arxiv.org/abs/cs/0504107
Description: Modern practical algorithms for shortest paths including bidirectional search and A* variants.
Why included: Bridge between theory and practice; efficiency considerations for real systems.
