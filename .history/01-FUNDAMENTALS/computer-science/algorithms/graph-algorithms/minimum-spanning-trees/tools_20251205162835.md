## BEGINNER

### VisuAlgo MST Visualizer
Creator: Steven Halim (National University of Singapore)
Year: 2011-Present
License: Creative Commons
Platform: Web-based (cross-platform)
Link: https://visualgo.net/en/mst
Language: JavaScript/HTML5
Description: Interactive visualization of Kruskal and Prim MST algorithms.
Why included: Educational standard; makes algorithm behavior clear through animation.

### NetworkX MST Functions
Creator: Python networkx contributors
Year: 2004-Present
License: BSD License (open source)
Platform: Python ecosystem
Link: https://networkx.org/documentation/stable/reference/algorithms/generated/networkx.algorithms.tree.mst.minimum_spanning_tree.html
Language: Python
Description: Production-ready MST algorithms (Kruskal, Prim) in Python.
Why included: Practical implementation; works with real network data.

## INTERMEDIATE

### Boost Graph Library - MST Algorithms
Creator: Boost C++ Community
Year: 1999-Present
License: Boost License (permissive open source)
Platform: C++ (cross-platform)
Link: https://www.boost.org/doc/libs/release/libs/graph/doc/
Language: C++
Description: High-performance Kruskal and Prim implementations in C++.
Why included: Performance-critical systems; production-grade code.

### Graphviz (Graph Visualization)
Creator: AT&T Labs Research
Year: 1991-Present
License: EPL 1.0 (open source)
Platform: Cross-platform
Link: https://graphviz.org/
Language: DOT language
Description: Tool for visualizing graph structures including MST results.
Why included: Visualization of MST results; shows graph topology clearly.
